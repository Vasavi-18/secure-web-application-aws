Secure Web Application on AWS

A Cloud Architecture Demonstration Project

Live deployment via AWS Application Load Balancer | Region: ap-south-2 (Hyderabad)

Access Application

This project demonstrates a secure multi-tier architecture hosted on AWS using EC2, Application Load Balancer, and RDS.

Architecture Components

AWS VPC

Logical isolated cloud network.

Public Subnet

Hosts internet-facing resources.

EC2 Instance

Compute server hosting the application.

Nginx Web Server

Handles HTTP requests.

Application Load Balancer

Distributes traffic across servers.

RDS MySQL Database

Managed relational database service.

Architecture Flow

🌐 Internet ⬇ ⚖ Application Load Balancer ⬇ 🖥 EC2 Instance (Nginx) ⬇ 🗄 RDS MySQL Database

Security Design

  • Database deployed in a private subnet.
  • Security Groups restrict database access.
  • Only EC2 instances can communicate with RDS.
  • Application Load Balancer handles external traffic.
  • Network isolation using AWS VPC.
